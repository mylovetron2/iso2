2. Vai trò & chức năng từng thành phần

  • index.php: Điều hướng đăng nhập/đăng xuất, chuyển hướng người dùng tới trang phù hợp.
  • projects.php: Router chính cho các thao tác CRUD Project.
  • admin_user_permissions.php: Giao diện phân quyền user (chỉ admin truy cập).
  • config/:
    • constants.php: Định nghĩa hằng số, quyền, role, require_once các file core.
    • database.php: Kết nối PDO tới MySQL.
  • controllers/:
    • ProjectController.php: Xử lý logic CRUD cho Project, nhận request từ projects.php.
  • includes/:
    • auth.php: Hàm xác thực, session, login/logout.
    • functions.php: Các hàm tiện ích chung.
    • permissions.php: Kiểm tra quyền, phân quyền động.
  • migrations/:
    • 20251011_users_stt_schema.sql: Tạo bảng users, roles, role_user, projects.
  • models/:
    • BaseModel.php: Model cơ sở thao tác DB.
    • Project.php: Model cho bảng projects.
    • User.php: Model cho bảng users, phân quyền.
  • views/:
    • admin/:
      • permissions_manager.php: Quản lý quyền cho role (admin UI).
      • user_permissions.php: Giao diện phân quyền user.
    • auth/:
      • login.php: Giao diện đăng nhập.
      • register.php: Giao diện đăng ký.
    • layouts/:
      • header.php: Sidebar, header, responsive, logic phân quyền UI.
      • footer.php: Footer chung.
    • projects/:
      • index.php: Giao diện danh sách, tìm kiếm, CRUD project.
